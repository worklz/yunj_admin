第一步：数据库配置
必填！数据库服务器地址，一般为127.0.0.1
必填！系统数据库端口，一般为3306
必填！系统数据库名,必须由字母/数字/下划线组合
必填！连接数据库的用户名
必填！连接数据库的密码
必填！建议使用默认,末尾字符必须为 '_'
第二步：账号配置
必填！Redis主机地址，一般为127.0.0.1
必填！Redis端口，一般为6379
必填！管理员账号最少4位
必填！保证密码最少6位
